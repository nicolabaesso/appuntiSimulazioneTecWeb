### Elencare gli accorgimenti che si devono utilizzare per rendere una tabella accessibile (domanda aperta)
* Dividere head e body (thead, tbody)
* Usare il tag scope con col, row, rowgroup, colgroup (o gli headers quando non si puó usare il tag scope)
* Alternare il colore di righe o colonne
* Scrivere la summary (xHTML) o aria-describedby (HTML5)
* Usare i tag lang e abbr quando necessario
* Non lasciare celle vuote (o almeno evitare)

### Qual è la differenza fondamentale tra gli attributi id e class? (completamento)
L'attributo id permette di identificare un solo tag. L'attributo class permette di identificare un gruppo di tag.\
Inoltre, un id puó essere utilizzato come link, cosa non possibile con una classe.

### Che cosa sono le convenzioni interne di un sito web e le convenzioni esterne? In cosa differiscono? (domanda aperta)

#### Convenzioni interne
Le convenzioni interne sono regole che l'utente impara all'interno del nostro sito

#### Convenzioni esterne
Convenzioni "universalmente riconosciute" es. link non visitati blu e link visitati viola

#### Differenze
Le convenzioni interne sostituiscono quelle esterne, e non devono essere cambiate una volta definite. Le convenzioni esterne possono essere rotte a patto di avere un tornaconto.

### Mettete in ordine di applicazione i seguenti stili CSS (in assenza di clausole !important) (scelta multipla)
In ordine di applicazione (dal primo applicato all'ultimo applicato):

1. Impostazioni di stile predefinite del browser
2. Fogli di stile esterni definiti dall’autore
3. Fogli di stile embedded definiti dall’autore
4. Impostazioni di stile inline definite dall’autore della pagina
5. Impostazioni personali dell’utente
